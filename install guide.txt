You will need python 3.7
A virtual environment is recomanded.

-Run the following cammands to setup:

conda create -n acrobot python=3.7
conda activate acrobot
pip install sklearn, xarray
pip install git+https://github.com/paris-saclay-cds/ramp-workflow.git@generative_regression_clean


-Create then a new directory, and clone the acrobot kit in it:


git clone https://github.com/ramp-kits/acrobot.git


-If you need pytorch:


conda install pytorch
